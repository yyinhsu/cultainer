# 隱私政策

**最後更新日期：** 2026 年 3 月 16 日

## 概述

Cultainer（以下簡稱「我們」）是一款個人閱聽紀錄與知識擷取工具。本隱私政策說明我們如何收集、使用及保護您的資訊。

## 我們收集的資訊

### 帳號資訊
- **Google 帳號資料**：當您使用 Google 登入時，我們會取得您的顯示名稱、電子郵件地址及大頭貼 URL。我們僅將這些資訊用於身份驗證及在應用程式中顯示您的個人資料。

### 使用者產生的內容
- **媒體紀錄**：您追蹤的書籍、電影、影集及音樂，包括標題、評分、評論及狀態。
- **書摘**：您儲存的文字摘錄、筆記及 AI 分析結果。
- **標籤**：您建立的自訂標籤，用於整理紀錄。

### 第三方 API 資料
- **Google Books / TMDB / Spotify**：當您搜尋媒體時，查詢會傳送至這些服務以取得中繼資料（標題、封面、簡介）。我們不會與這些服務分享您的個人資料。
- **Google Gemini API**：當您使用 AI 功能（分析、摘要、評論增強）時，您的摘錄文字會傳送至 Google 的 Gemini API 進行處理。您的 API 金鑰僅儲存於您的裝置本機。
- **Google ML Kit**：OCR 文字辨識在裝置上處理，不會將影像資料傳送至外部伺服器。

## 我們如何儲存您的資料

- 所有使用者資料（紀錄、書摘、標籤、個人資料）儲存於 **Google Firebase**（Firestore），歸屬於您的已驗證帳號之下。
- 資料透過 Firebase 在您的裝置間同步。
- 您的 **Gemini API 金鑰**使用 SharedPreferences 儲存於裝置本機，絕不會上傳至我們的伺服器。

## 資料分享

我們**不會**：
- 將您的個人資料出售給第三方
- 與廣告商分享您的資料
- 將您的資料用於提供應用程式功能以外的目的

我們**會**與以下服務分享資料：
- **Google Firebase**：用於身份驗證及資料儲存（受 [Google 隱私政策](https://policies.google.com/privacy)規範）
- **Google Gemini API**：僅在您主動觸發 AI 功能時，且僅限您選擇分析的文字

## 資料保留

- 您的資料在您維持有效帳號期間保留。
- 您可以隨時在應用程式中刪除個別紀錄、書摘及標籤。
- 若要刪除整個帳號及所有相關資料，請透過以下方式聯繫我們。

## 安全性

- 所有資料傳輸均使用 HTTPS 加密。
- Firebase 安全規則限制資料存取，僅允許已驗證的使用者檢視自己的資料。
- 我們遵循業界標準的安全措施。

## 兒童隱私

Cultainer 並非針對 13 歲以下兒童設計。我們不會故意收集 13 歲以下兒童的個人資訊。

## 政策變更

我們可能會不定期更新本隱私政策。任何變更將透過在應用程式內發布新的隱私政策通知您。

## 聯繫我們

如您對本隱私政策有任何疑問，請在我們的 GitHub 儲存庫提出 issue 或透過電子郵件聯繫我們。

*本隱私政策適用於 Cultainer 行動及桌面應用程式。*
