# Asset Directories

This folder contains application assets.

## Structure

- `images/` - Image assets (PNG, JPG, WebP)
- `icons/` - SVG icon assets
- `fonts/` - Custom font files

## Required Fonts

Download and add the following font files:

### Fraunces (Headers)
- `Fraunces-SemiBold.ttf` (weight: 600)

### DM Sans (Body)
- `DMSans-Regular.ttf` (weight: 400)
- `DMSans-Medium.ttf` (weight: 500)
- `DMSans-SemiBold.ttf` (weight: 600)

### Inter (System/Numbers)
- `Inter-Regular.ttf` (weight: 400)
- `Inter-Medium.ttf` (weight: 500)

Download from:
- Fraunces: https://fonts.google.com/specimen/Fraunces
- DM Sans: https://fonts.google.com/specimen/DM+Sans
- Inter: https://fonts.google.com/specimen/Inter
